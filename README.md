# Lingo Landing Page

Официальный веб-сайт и промо-страница для приложения-переводчика **Lingo**. 

Сайт разработан для презентации возможностей переводчика, привлечения пользователей и сбора заявок на ранний доступ с бесплатным 3-месячным триал-периодом.

## 🚀 О проекте Lingo
**Lingo** — это современное приложение-переводчик с единой системой авторизации между сайтом и десктопным приложением. Пользователи могут зарегистрироваться прямо здесь, на сайте, получить промо-период и сразу использовать свой аккаунт в приложении.

## 🛠 Технологический стек сайта
* **Фронтенд:** HTML5, CSS3 (Кастомная дизайн-система с поддержкой тем)
* **Хостинг и автоматизация:** Готов к деплою на Netlify
* **Формы и бэкенд-логика:** Интеграция с Resend для безопасной обработки заявок и подписок
