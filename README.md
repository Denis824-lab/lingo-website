# Lingo Landing Page

Официальный веб-сайт и промо-страница для приложения-переводчика **Lingo**. 

Сайт разработан для презентации реальных возможностей переводчика, регистрации единого аккаунта и скачивания бесплатной открытой beta-версии для Windows.

## О проекте Lingo
**Lingo** — это современное приложение-переводчик с единой системой авторизации между сайтом и десктопным приложением. Пользователи могут зарегистрироваться на сайте, подтвердить email и использовать тот же аккаунт в приложении. Во время открытой beta все функции доступны бесплатно.

## Технологический стек сайта
* **Фронтенд:** HTML5, CSS3 (Кастомная дизайн-система с поддержкой тем)
* **Хостинг и автоматизация:** Готов к деплою на Netlify
* **Формы и бэкенд-логика:** Интеграция с Resend для безопасной обработки заявок и подписок

## Локальный запуск

Откройте `index.html` в браузере для проверки интерфейса. Для проверки отправки формы установите Netlify CLI и запустите:

```powershell
npx netlify dev
```

## Настройка подписки

1. В [Resend](https://resend.com) создайте API-ключ и подтвердите домен отправителя.
2. Скопируйте `.env.example` в `.env` и задайте значения переменных.
3. В Netlify добавьте те же переменные окружения для Production и Deploy Previews:
   - `SITE_ORIGIN` — публичный адрес сайта без завершающего `/`;
   - `RESEND_API_KEY` — API-ключ Resend;
   - `RESEND_FROM_EMAIL` — подтверждённый адрес отправителя Resend;
   - `RESEND_TO_EMAIL` — адрес, на который команда получает заявки.
4. Разверните репозиторий в Netlify. Конфигурация `netlify.toml` автоматически публикует сайт и функции:
   - `/.netlify/functions/subscribe` — подписка на ранний доступ;
   - `/.netlify/functions/contact` — отправка сообщений с формы обратной связи.

Секреты находятся только в переменных окружения хостинга. Файл `.env` исключён из Git.

Пока в Resend не подтверждён собственный домен, сервис принимает тестовые письма только на email владельца аккаунта. В этом режиме заявка приходит на `RESEND_TO_EMAIL`, а адрес посетителя указан в тексте письма. После подтверждения домена можно добавить `reply_to` в `netlify/functions/subscribe.mjs`.

## Настройка аналитики

### Яндекс.Метрика

1. Создайте счётчик в [Яндекс.Метрике](https://metrika.yandex.ru).
2. В `index.html` найдите скрипт и замените `XXXXXXXX` на номер вашего счётчика:
   ```javascript
   ym(XXXXXXXX, "init", { ... });
   ```

### Cookie Consent

Сайт включает механизм согласия на использование cookie. Аналитика загружается только после явного согласия пользователя. Согласие сохраняется в `localStorage`.

## SEO

Сайт включает:
- Open Graph и Twitter Cards мета-теги для превью в соцсетях;
- JSON-LD структурированные данные (Schema.org SoftwareApplication);
- `sitemap.xml` — карта сайта;
- `robots.txt` — правила для поисковых роботов;
- `<link rel="canonical">` — канонические URL.

**Важно:** Замените `https://lingo.app/og-image.jpg` на реальное изображение для превью (рекомендуемый размер: 1200x630 px).
