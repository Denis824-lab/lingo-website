# Lingo landing

Статический промо-сайт Lingo с безопасной подпиской на ранний доступ. Каждая заявка доставляется на почту команды через Resend.

## Локальный запуск

Откройте `index.html` в браузере для проверки интерфейса. Для проверки отправки формы установите Netlify CLI и запустите:

```powershell
npx netlify dev
```

## Настройка подписки

1. В [Resend](https://resend.com) создайте API-ключ и подтвердите домен отправителя.
2. Скопируйте `.env.example` в `.env` и задайте значения переменных.
3. В Netlify добавьте те же переменные окружения для Production и Deploy Previews:
   - `SITE_ORIGIN` — публичный адрес сайта без завершающего `/`;
   - `RESEND_API_KEY` — API-ключ Resend;
   - `RESEND_FROM_EMAIL` — подтверждённый адрес отправителя Resend;
   - `RESEND_TO_EMAIL` — адрес, на который команда получает заявки.
4. Разверните репозиторий в Netlify. Конфигурация `netlify.toml` автоматически публикует сайт и функцию `/.netlify/functions/subscribe`.

Секреты находятся только в переменных окружения хостинга. Файл `.env` исключён из Git.

Пока в Resend не подтверждён собственный домен, сервис принимает тестовые письма только на email владельца аккаунта. В этом режиме заявка приходит на `RESEND_TO_EMAIL`, а адрес посетителя указан в тексте письма. После подтверждения домена можно добавить `reply_to` в `netlify/functions/subscribe.mjs`.

## Перед публикацией

- замените ссылку GitHub на действующую;
- добавьте юридически утверждённую политику конфиденциальности и ссылку на неё;
- замените превью в первом экране на реальный скриншот приложения;
- настройте домен `lingo.app` и укажите его в `SITE_ORIGIN`.
