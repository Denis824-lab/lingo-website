# Lingo Landing Page

Официальный веб-сайт и промо-страница для приложения-переводчика **Lingo**. 

Сайт разработан для презентации реальных возможностей переводчика, регистрации единого аккаунта и скачивания бесплатной открытой beta-версии для Windows.

## О проекте Lingo
**Lingo** — это современное приложение-переводчик с единой системой авторизации между сайтом и десктопным приложением. Пользователи могут зарегистрироваться на сайте, подтвердить email и использовать тот же аккаунт в приложении. Во время открытой beta все функции доступны бесплатно.

## Технологический стек сайта
* **Фронтенд:** HTML5, CSS3 (Кастомная дизайн-система с поддержкой тем)
* **Хостинг:** GitHub Pages из ветки `main`
* **Авторизация:** Firebase Auth (email/password и Google)
* **Обратная связь:** безопасная подготовка письма через локальный почтовый клиент

## Локальный запуск

Откройте `index.html` в браузере или запустите любой локальный статический сервер, например:

```powershell
python -m http.server 8080
```

## Публикация

GitHub Pages публикует корень ветки `main`. Файл `CNAME` подключает домен `lingo-app.ru`. Для apex-домена используются четыре стандартных A-записи GitHub Pages, а `www` направляется CNAME-записью на `denis824-lab.github.io`.

## Настройка аналитики

### Яндекс.Метрика

1. Создайте счётчик в [Яндекс.Метрике](https://metrika.yandex.ru).
2. В `index.html` найдите скрипт и замените `XXXXXXXX` на номер вашего счётчика:
   ```javascript
   ym(XXXXXXXX, "init", { ... });
   ```

### Cookie Consent

Сайт включает механизм согласия на использование cookie. Аналитика загружается только после явного согласия пользователя. Согласие сохраняется в `localStorage`.

## SEO

Сайт включает:
- Open Graph и Twitter Cards мета-теги для превью в соцсетях;
- JSON-LD структурированные данные (Schema.org SoftwareApplication);
- `sitemap.xml` — карта сайта;
- `robots.txt` — правила для поисковых роботов;
- `<link rel="canonical">` — канонические URL.

**Важно:** Замените `https://lingo.app/og-image.jpg` на реальное изображение для превью (рекомендуемый размер: 1200x630 px).

## Юридические документы

- `privacy.html` — политика конфиденциальности (соответствует 152-ФЗ);
- `terms.html` — условия использования.

**Важно:** Тексты документов являются шаблонами. Рекомендуется показать их юристу перед публикацией.
